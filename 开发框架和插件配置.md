# 1.安装vue2.0

```js
// 以下是npm终端

# 使用vue-cll脚手架安装vue -   router vuex 已选安装
vue reacte purchase
```

# 2.安装vant 框架

```js
// 以下是npm终端
# 通过 npm 安装
npm i vant -S
```

# 3. (废弃)安装   babel-plugin-import

```js
## 以下已废弃
## 以下已废弃
## 以下已废弃

// vant自动按需引入组件
# 安装插件
npm i babel-plugin-import -D


// 对于使用 babel7 的用户，可以在 babel.config.js 中配置
module.exports = {
  plugins: [
    ['import', {
      libraryName: 'vant',
      libraryDirectory: 'es',
      style: true
    }, 'vant']
  ]
};
// 接着你可以在代码中直接引入 Vant 组件
// 插件会自动将代码转化为方式二中的按需引入形式
import { Button } from 'vant';
```

# 4.vant全局引入 导入所有组件

```js
Vant 支持一次性导入所有组件

import Vue from 'vue';
import Vant from 'vant';
import 'vant/lib/index.css';

Vue.use(Vant);
```
